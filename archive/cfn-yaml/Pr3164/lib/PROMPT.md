Static Website Deployment using AWS CloudFormation with YAML in Single Output File
• Deploy a static website using AWS CloudFormation.
• Host website content in an S3 bucket.
• Set up CloudFront distribution with SSL (HTTPS) for secure and low-latency delivery.
• Configure Route 53 for custom domain and DNS management.
• Apply an S3 bucket policy to allow public read access to website content.
• Enable CloudWatch metrics/logging to track traffic and performance.
• Use AWS KMS for encrypting S3 data at rest.
• Ensure the design is simple, cost-effective, and can support ~5,000 daily visitors.
