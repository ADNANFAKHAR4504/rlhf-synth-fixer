# Task: Aurora Global Database for Cross-Region Disaster Recovery

Create a CloudFormation template to deploy an Aurora Global Database for cross-region disaster recovery.

## Requirements

- Deploy an Amazon Aurora Global Database
- Configure cross-region replication for disaster recovery
- Ensure high availability and automatic failover capabilities
- Follow AWS best practices for Aurora Global Database deployment
- Include necessary security configurations (VPC, Security Groups, IAM roles)
- Implement monitoring and alerting for the global database

## Deliverables

- CloudFormation template (JSON format)
- Infrastructure code following best practices
- Proper resource naming and tagging
- Documentation of the architecture and configuration choices
